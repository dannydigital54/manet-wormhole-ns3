# MANET Wormhole NS-3

## Description
This repository contains a reproducible NS-3 simulation for a MANET wormhole attack scenario.

## Parameters
- 30 nodes
- Area: 80x80
- AODV routing
- 5 UDP flows at 256 Kbps

## Execution (Windows)
Run:
run.bat

## Notes
This scenario is illustrative and supports a research study.
